# 侧边栏布局设计

## 概述

将 Sana 从单栏布局改为双栏布局：左侧固定侧边栏展示统计和热力图，右侧主内容区承载搜索、输入和笔记列表。参考 flomo 的简洁风格。

## 布局结构

```
+------------------+----------------------------------------+
|   侧边栏 (240px) |           主内容区 (自适应)              |
|                  |                                        |
|  📊 统计信息      |  [🔍 搜索] [📤] [📥]                    |
|  - 12 笔记        |                                        |
|  - 5 天          |  [========== 笔记输入框 ==========]       |
|                  |                                        |
|  热力图           |  [今天]                                |
|  一月 二月 三月   |    - 笔记1                              |
|  ▪️ ▪️ ▪️ ▪️     |    - 笔记2                              |
|  ▪️ ▪️ ▪️ ▪️     |  [昨天]                                |
|  ▪️ ▪️ ▪️ ▪️     |    - 笔记3                              |
|                  |  ...                                    |
+------------------+----------------------------------------+
```

## 侧边栏

**统计信息：**
- 笔记总数
- 记录天数（首次创建笔记距今的天数）

**热力图：**
- 显示近 3 个月的活动
- 颜色深浅 = 当日创建笔记数量（0/1/2/3+ 四个等级）
- 紧凑网格（每日一个小方块）

## 主内容区

与现有 TimelineView 保持一致：
- 顶部操作栏：搜索（打开 Modal）、导出、导入
- 笔记输入框（MemoComposer）
- 时间分组笔记列表（TimeGroup + MemoCard）
- 加载更多

## API 变更

**新增端点：GET /api/memos/stats**

返回：
```json
{
  "memo_count": 12,
  "active_days": 5,
  "heatmap": {
    "2026-04-01": 2,
    "2026-03-30": 1
  }
}
```

用于渲染侧边栏统计和热力图。

## 组件变更

| 组件 | 操作 |
|------|------|
| `Layout.vue` | 新增：侧边栏 + 主内容区双栏布局 |
| `Sidebar.vue` | 新增：统计数字 + 热力图 |
| `Heatmap.vue` | 新增：GitHub 风格热力图 |
| `TimelineView.vue` | 移除头部，保留搜索/导入/导出 + 笔记列表 |
| `App.vue` | 路由指向 Layout 而非直接 TimelineView |

## 热力图颜色

- 无活动：`#ebedf0`（浅灰）
- 1条：`#9be9a8`（浅绿）
- 2条：`#40c463`（中绿）
- 3+条：`#30a14e`（深绿）
